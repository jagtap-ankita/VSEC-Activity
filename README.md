# 🚀 Startup Success Prediction using Machine Learning

## 📌 Project Overview

This project predicts the success of startups using machine learning techniques based on financial, investment, and growth-related features.

---

## 📊 Dataset

The dataset includes important features such as:

* Industry
* Funding Rounds
* Investment Amount (USD)
* Valuation (USD)
* Number of Investors
* Country
* Year Founded
* Growth Rate (%)

---

## 🤖 Machine Learning Models Used

* Random Forest
* Support Vector Machine (SVM)
* Naive Bayes
* Logistic Regression
* K-Nearest Neighbors (KNN)

---

## 🏆 Results

* Best Performing Model: **K-Nearest Neighbors (KNN)**
* Accuracy Achieved: **96.97%**

---
